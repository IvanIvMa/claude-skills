# Claude Skills Collection

Custom Skills für Claude — Multi-Perspektiven-Beratungsframeworks.

## 🏛️ The Council

Vollständiges Beratungsgremium mit 5 unabhängigen KI-Beratern:
- 🎯 **Der Stratege** — Langfristige Positionierung
- 🔍 **Der Skeptiker** — Risiken & versteckte Kosten
- 🔧 **Der Pragmatiker** — Umsetzbarkeit & Ressourcen
- 🚀 **Der Visionär** — Innovation & Disruption
- 👥 **Der Menschenkenner** — Stakeholder & Teamdynamik

Inkl. Peer Review, Chairman-Synthese und Minority Report.
Adaptiert von Andrej Karpathys LLM Council.

**Trigger:** "Convene the Council", "Council einberufen", "Stress-test this"

## 💡 Der Berater

Schnelle Drei-Perspektiven-Beratung:
- 🔥 **Der Kritiker** — Findet jede Schwachstelle
- ✨ **Die Visionärin** — Sieht das 10x-Potenzial
- ⚖️ **Der Realist** — Gibt den konkreten Aktionsplan

Leichtere, schnellere Alternative zum Council für alltägliche Entscheidungen.

**Trigger:** "Der Berater", "Schnelle Einschätzung", "Bewerte meine Idee"

## Installation

Die `.skill`-Dateien können direkt in Claude installiert werden.
